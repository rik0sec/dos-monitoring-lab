🔴 Denial of Service Lab — Port 5004

1. Server (Perangkat A): 
python website.py
2. Attacker (Perangkat B): 
python attack.py
⚙ Simulasi Beda Perangkat:

Di website.py baris terakhir, ubah host:
app.run(host="0.0.0.0", port=5004, threaded=True)

Di attack.py baris BASE_URL, ubah IP:
BASE_URL = "http://<IP_SERVER>:5004"

Cek IP server: ifconfig (Linux/Mac) atau ipconfig (Windows)
